# Flutter Weather AI
[![](https://img.shields.io/badge/Flutter-0553B1?style=for-the-badge&logo=flutter&logoColor=white)](https://flutter.dev/)
[![](https://img.shields.io/badge/Cursor-14120C?style=for-the-badge&logo=cursor&logoColor=white)](https://cursor.com/)

This repository centralizes the development of a Weather Forecast App in Flutter, designed as a practical project to demonstrate **AI-Driven Development** using modern tooling.

The application is built leveraging **Cursor** as the primary AI coding agent, enabling rapid development with minimal manual coding while maintaining high standards of code quality and scalability.

The project follows **Clean Architecture** principles and enforces strict development guidelines ("Cursor Rules") to ensure maintainability, type safety, and clear separation of concerns, inspired by best practices from the Flutter community.

## Stack

| Category | Technologies |
|-----------|------------|
| **Language** | Dart (Null Safety) |
| **Architecture** | Clean Architecture |
| **State Management** | Flutter Bloc / Cubit |
| **Networking** | http |
| **Data Modeling** | Freezed, Json Serializable |
| **AI Coding Assistants** | Cursor |
| **API Integration** | Weather API |
